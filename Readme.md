# Code Review Application e-commerce Spring Boot

Vous trouverez ci-joint un ensemble de fichiers Java constituant une application Spring Boot de e-commerce très basique.

Le code compile, mais il contient volontairement plusieurs problèmes de niveaux variés.

🚀 Votre mission est de relire ces fichiers et de noter vos observations à l'écrit :

- Identifiez les problèmes que vous repérez
- Expliquez pourquoi c'est problématique
- Proposez des améliorations

🔍 Les problèmes à déceler dans le code concernent :

- La **qualité et la lisibilité du code**
- La **conception et l'architecture applicative**
- La **stratégie de test**
- Les **performances et l'utilisation de JPA**

_Nous n'attendons pas de vous que vous trouviez tous les problèmes !_

L'objectif est d'observer votre manière de lire du code, votre capacité à le comprendre, et de voir ce qu'il est important de corriger selon vous et pourquoi.

⏳ Le timing de l'exercice sera le suivant :

- 30 à 40 minutes pour les 3 premières sections
- 5 minutes pour des questions
- 15 minutes pour les aspects performances et utilisation de JPA (selon timing)
